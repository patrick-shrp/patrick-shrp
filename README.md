### AUVY — science of execution
Operating substrate for knowledge work: **Cortex** (consulting intelligence with receipts), **Synapse** (GenOps runtime), evidence layer, venture spawn.

`→` [auvy.ai](https://www.auvy.ai) · [cortex.auvy.ai](https://cortex.auvy.ai) · [pricing](https://www.auvy.ai/pricing)

### ImTakt — mobility for agents
Zero-config MCP / SDK for Deutsche Bahn–scale transit. Journeys, stops, realtime — built for operators and agents.

`→` [imtakt.dev](https://imtakt.dev) · [try](https://imtakt.dev/try)

### OpenPulse — wire between worlds
Encrypted envelopes for humans + agents. Protocol company, not another chat app.

`→` [openpulse.space](https://openpulse.space)
