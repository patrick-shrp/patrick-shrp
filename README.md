## Building

### AUVY — science of execution
The substrate for knowledge work: **Cortex** (guardrailed consulting intelligence), **Synapse** (GenOps runtime), evidence layer, venture spawn. Autonomy with proof.

`→` [auvy.ai](https://www.auvy.ai) · [cortex.auvy.ai](https://cortex.auvy.ai) · [pricing](https://www.auvy.ai/pricing)

### ImTakt — mobility that agents can run
Zero-config MCP / SDK for Deutsche Bahn–scale transit. A system that keeps answering when operators move on.

`→` [imtakt.dev](https://imtakt.dev) · [try](https://imtakt.dev/try)

### OpenPulse — wire between organizations
Encrypted envelopes for humans + agents. Inter-org autonomy needs a real protocol — not another inbox.

`→` [openpulse.space](https://openpulse.space)

### ImStudium — study systems that keep running
Stud.IP sync → slides/PDF corpus → Klausur planning. CLI + MCP so agents work the corpus without babysitting.

`→` [github.com/patrick-shrp/imstudium](https://github.com/patrick-shrp/imstudium)
